<!--more-->

几个音（一般不超过七个，不少于三个）按照一定的关系（高低关系、稳定与不稳定的关系）联结在一起，构成一个音组织，并以某一个音为中心。这个音组织就称为**调式**。

在这个概念中，有一句话值得深入理解：“以某一个音为中心”。为了理解这句话，我们需要先了解什么是**调**。

**调**是由基本音级所构成的音序列的音高位置。用更加通俗易懂的话来说，do 的音高位置叫做**调**。

而**调式**就是这样一个音组织，它需要以某一个音为中心，这就是**调**。这个中心音就是**主音**。此外，还有一些音充当稳定音，它们出现的频率相对较高；而还有一些音充当不稳定音，它们出现的频率相对较低。所有的这些音结合在一起，就构成了一个调式。

在东西方的体系中，调试的分类截然不同。在西方的体系中，调试被分为大调和小调。而在我国的民族音乐体系中，调试是以“宫商角徵羽”为基础的。

## 西方调式

在西方的体系中，调试被分为大调和小调。大调和小调的区别在于，大调和是主音和中音之间是大三度，而小调是主音和中音之间是小三度。

### 自然大调

自然大调的音阶，即前文的口诀“全全半全全全半”。例如 C4-D4-E4-F4-G4-A4-B4-C5，即我们耳熟能详的“do-re-mi-fa-sol-la-si-do”。

自然大调是一种明亮、欢快的调式，广泛应用于各种音乐风格中。

调式中的每个音都有一个自己特有的名称和号数，如下表所示：

| 音名 | 号数 |  名称  |
| :--: | :--: | :----: |
|  do  |  I   |  主音  |
|  re  |  II  | 上主音 |
|  mi  | III  |  中音  |
|  fa  |  IV  | 下属音 |
| sol  |  V   |  属音  |
|  la  |  VI  | 下中音 |
|  si  | VII  |  导音  |

这里的每一级音的命名是有其特殊含义的。I 级音最稳定，所以命名为**主音**；按照五度相生律，向上五度得到 V 级音，之所以命名为**属音**，是因为它是从属于主音的；同理，按照五度相生律，向下五度得到 IV 级音，所以命名为**下属音**；III 级音正好在主音和属音的中间，所以命名为**中音**；同理，VI 级音命名为**下中音**；II 级音为从主音开始往上数的一级，所以命名为**上主音**；而 VII 级音在主音之下一级，不过它与主音之间形成小二度，具有强烈地想要回到主音的倾向，所以命名为**导音**。

### 自然小调

自然小调的音阶是“全半全全半全全”。例如 A4-B4-C4-D4-E4-F4-G4-A5，即“la-si-do-re-mi-fa-sol-la”。

自然小调是一种较为忧郁、深沉的调式，常用于表达悲伤、哀怨等情感。

{{< callout type="info" >}}

为什么大调使人感到欢快，而小调使人感到忧郁呢？

这其实是因为大调和小调的音程不同。大调的主音和中音之间是大三度，而小调的主音和中音之间是小三度。

{{< /callout >}}

类似地，自然小调的每一级音也有自己的名称和号数，如下表所示：

| 音名 | 号数 |  名称  |
| :--: | :--: | :----: |
|  la  |  I   |  主音  |
|  si  |  II  | 上主音 |
|  do  | III  |  中音  |
|  re  |  IV  | 下属音 |
|  mi  |  V   |  属音  |
|  fa  |  VI  | 下中音 |
| sol  | VII  |  导音  |

{{< callout type="warning" >}}

**调和主音的区别与联系**

调和主音这两个概念极易混淆。“调”是指一个调式中 do 所指向的音名，而“主音”是指这个调式中最稳定的音，即 I 级音。

大调的调和主音相同，但是小调的调和主音不同。例如 C 大调的调为 C（或者说 C 大调属于 C 调），且 C 是 C 大调的主音。而 a 小调的调为 C（也就是说 a 小调也属于 C 调，这一点可以通过五线谱中共用同一套升降号标记来判断），但是 A 是 a 小调的主音。

{{< /callout >}}

### 自然小调的变体

#### 和声小调

{{< abcjs id="minor-chord" >}}

X: 1
T: 和声小调
M: 4/4
L:1/8
V:1
A,B,CD EF^GA | A^GFE DCB,A, ||

{{< /abcjs >}}

将自然小调的第 VII 级音升高半音，得到的就是和声小调。例如，将 A3-B3-C4-D4-E4-F4-G4-A4 中的 G4 升高半音，得到 A3-B3-C4-D4-E4-F4-G#4-A4。

为了使小调的导音到主音能有和大调一样的倾向性，即相差小二度，所以将小调的第七级音升高半音，得到和声小调。所以和声小调不是自然调式，而是人为定义的。由于导音的升高，导致 fa 和 sol 之间相差增二度，这会导致歌曲不容易唱。因此，就有了[旋律小调](./#旋律小调)。

#### 旋律小调

{{< abcjs id="melodic-minor" >}}

X: 1
T: 旋律小调
M: 4/4
L:1/8
V:1
A,B,CD E^F^GA | A^GFE DCB,A, ||

{{< /abcjs >}}

将和声小调的第 VI 级音升高半音，得到的就是旋律小调。例如，将 A3-B3-C4-D4-E4-F4-G#4-A4 中的 F4 升高半音，得到 A3-B3-C4-D4-E4-F#4-G#4-A4。而下行时，F#4 还原为 F4，即 A4-G#4-F4-E4-D4-C4-B3-A3。

旋律小调在音乐中常用于表现较为复杂的情感，其上行音阶的升高音级赋予了旋律一种向上的张力和明亮感，而下行时还原的音级则回归到自然小调的柔和与忧郁，给人一种古典、柔和的感觉。

## 中国民族调式

### 五声调式

按照纯五度排列起来的五个音所构成的调式，由低到高分别命名为宫、商、角、徵、羽。这个是古代的音律，类似于现代的“do-re-mi-sol-la”。

在很多中文歌曲中，几乎听不到“fa”和“ti”这两个音，就是因为它们采用的是五声调式，“fa”和“ti”这两个音在中国民族音乐中属于**偏音**。而且，在这些中文歌曲中，如果引入了“fa”和“ti”这两个音，我们直观上会觉得整首歌曲的旋律变得奇怪，没有了古风的韵味。

如果五声调式以宫音作为主音，就称为宫调式。同理可根据主音的不同，得到五种不同的五声调式，即宫调式、商调式、角调式、徵调式、羽调式。

在钢琴上，所有的黑键构成五声调式。

### 同宫系统各调

宫音相同的各调式，称为同宫系统各调。要找出同宫系统各调，只要按照 do-re-mi-sol-la 的音程关系，找出相应的音名即可。

| 系统 | 宫调式 | 商调式 | 角调式 | 徵调式 | 羽调式 |
| :--: | :----: | :----: | :----: | :----: | :----: |
|  C   |   C    |   D    |   E    |   G    |   A    |
|  D   |   D    |   E    |   F#   |   A    |   B    |
|  E   |   E    |   F#   |   G#   |   B    |   C#   |
|  A   |   A    |   B    |   C#   |   E    |   F#   |
|  B   |   B    |   C#   |   D#   |   F#   |   G#   |

### 五声调式的变体

**六声调式**是指在五声调式的基础上，添加 fa 这个偏音所形成的调式。具体来说，包括加入清角（fa）和变徵（升 fa）两种情况。

如果在六声调式的基础上，继续添加 si 这个偏音，就得到了**七声调式**。具体来说，包括加入闰（降 si）和变宫（si）两种情况。七声调式在中国民族音乐中有着独特的分类体系：

1. 加入清角（fa）和变宫（si），得到雅乐调式音阶。
2. 加入清角（fa）和闰（降 si），得到清乐调式音阶。
3. 加入变徵（升 fa）和闰（降 si），得到燕乐调式音阶。

{{< callout type="info" >}}

**偏音的命名规则**

七声调式是在五声基础上，为旋律需要引入的偏音。而偏音的命名不是按绝对音高，而是按功能与归属关系来命名的。它讲究如下三个原则：

- 以五声正音为参照：名字说明它“靠近哪一个正音”
- “变”表示从正音派生：功能上隶属于某个正音（不强调升降记号）
- “清”、“闰”是功能词而非音高词：功能上不隶属于任何正音
  - 清：较高、较明亮的同类音
  - 闰：为补足体系而插入的过渡音

所以，引入的四个偏音可以这样理解：

| 偏音 | 近似音高 | 命名含义概括             |
| ---- | -------- | ------------------------ |
| 清角 | fa       | 比角更高、更清的角类音   |
| 变徵 | 升 fa    | 向下变化、功能接近徵的音 |
| 闰   | 降 si    | 体系外插入的补充过渡音   |
| 变宫 | si       | 向下变化、指向宫的音     |

{{< /callout >}}

## 调式判断

调式判断是分析和研究任何一首音乐作品的基本方法。它的目标是确定这首作品采用的是哪一种调式。它的一般步骤如下：

{{% steps %}}

### 看调号

- 对于简谱，“1=X”就表示它是 X 调。
- 对于五线谱，根据升降号数量判断。

### 看尾音

结束音通常是主音。

- 对于西方音乐，可以据此判断大调还是小调。
- 对于中国民族音乐，可以判断属于宫商角徵羽的哪一种。

### 看线条

- 中国民族音乐的特点：接二连三，绕来绕去
- 西方音乐的特点：循序渐进，跳来跳去

### 看素材

即看包含了哪些音。

- 对于西方音乐，可以进一步确认是不是自然大小调的变体。
- 对于中国民族音乐，可以判断是否为五声调式的变体。

{{% /steps %}}

举例说明：

如果作品中只出现了 do-re-mi-sol-la 这五个音，那么就是五声调式。do 的音名如果是 D，那么就是 D 五声宫调式；do 的音名如果是 E，那么就是 E 五声宫调式，以此类推。

如果作品中出现了 do-re-mi-fa-sol-la-si 这七个音，那么有两种可能：

1. 若七个音具有同等的重要性，那就是自然大调。do 的音名如果是 D，那么就是 D 自然大调；do 的音名如果是 E，那么就是 E 自然大调，以此类推。
2. 若七个音中主要是 do-re-mi-sol-la 这五个音，fa 和 si 只是偶尔出现，那就是七声宫调式。do 的音名如果是 D，那么就是 D 七声宫调式；do 的音名如果是 E，那么就是 E 七声宫调式，以此类推。

如果结尾音不是 do，那么就需要根据结尾音的音名，以及作品中的其他音，来判断调式。

对于五声调式的情况，结尾音如果是 re，那就是五声商调式；如果是 mi，那就是五声角调式；如果是 sol，那就是五声徵调式；如果是 la，那就是五声羽调式。

对于结尾音是 la，且作品中出现了 do-re-mi-fa-sol-la-si 这七个音的情况，如果这七个音具有同等的重要性，那就是自然小调；如果这七个音中主要是 do-re-mi-sol-la 这五个音，fa 和 si 只是偶尔出现，那就是七声羽调式。

## 调式变音

将自然调式中的音加以升高或降低而得的音，叫做**调式变音**。前文所述的[和声小调](./#和声小调)和[旋律小调](./#旋律小调)就是调式变音的例子。

调式变音的产生，一定是在构成全音的两个音之间。调式变音的产生，只是加强原来的倾向性，并不改变调式音级的基本作用。通常调式变音所使用的变音记号，只能临时记在音符的前面。
