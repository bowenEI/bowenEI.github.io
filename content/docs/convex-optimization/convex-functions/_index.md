## 本章涉及到的数学符号

| 符号                   | 含义                      |
| ---------------------- | ------------------------- |
| \(\tilde{f}\)            | 函数 \(f\) 的扩展值延伸     |
| \(C_\alpha\)             | 函数的 \(\alpha\)- 下水平集 |
| \(\operatorname{epi} f\) | 函数 \(f\) 的上镜图         |
| \(h \circ g\)            | 复合函数 \(h(g)\)           |
| \(\inf f\)               | 函数 \(f\) 的下确界         |
| \(f^{*}\)               | 函数 \(f\) 的共轭函数       |
